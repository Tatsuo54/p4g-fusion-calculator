# P4G ペルソナ合体検索ツール

ペルソナ4 ザ・ゴールデンの合体検索ツールです。

## 機能

- **順引き検索**: 素材ペルソナを選んで合体結果を確認
- **逆引き検索**: 作りたいペルソナから必要な素材を検索
- **対応合体**: 2身合体、3身合体、特殊合体

## 開発

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```

## デプロイ

Cloudflare Pagesにデプロイする場合:
- ビルドコマンド: `npm run build`
- 出力ディレクトリ: `dist`
